# Media Foundation Tool

判断Dolby atmos是否启用的工具

## 使用方法

```
MFTool.exe
```

观察输出结果的GUID是否存在

```
Subtype: {FA39CD94-BC64-4AB1-9B71-DCD09D5A7E7A}
```

如果存在，则为支持Dolby Atmos
